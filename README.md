# TypeScript Practice & Sandbox

A central repository for my TypeScript learning journey, featuring exercises from books, competitive type challenges, and experimental code snippets.

The goal of this repo is to deepen my understanding of TypeScript's type system, advanced types, and frontend applications.

## 📂 Repository Structure

*   📁 `programming-typescript-book/` – Exercises and notes from *Programming TypeScript* by Boris Cherny.
*   📁 `type-challenges/` – My solutions to the popular open-source [type-challenges](https://github.com/type-challenges/type-challenges) repository.
*   📁 `snippets/` – Miscellaneous utility functions, playground experiments, and quick TypeScript recipes.

## 🚀 Technical Goals

*   Master advanced types (Conditional types, Mapped types, Template Literal types).
*   Improve type safety in day-to-day frontend development.
*   Explore TypeScript compiler configurations (`tsconfig.json` optimization).
